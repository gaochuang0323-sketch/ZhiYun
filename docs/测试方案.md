# ZhiYun BMS 故障模拟器 — 完整测试方案

## 1. 概述

本文档为 ZhiYun BMS 故障模拟器固件项目提供完整的单元测试方案，基于 Ceedling + Unity + CMock + gcov 测试框架。

### 1.1 测试框架

| 组件 | 版本/说明 |
|------|----------|
| Ceedling | 1.0.1 (gem) |
| Unity | 内置 |
| CMock | 内置，支持 ignore/callback/return 插件 |
| gcov | gcovr 8.6，生成 HTML 报告 |

### 1.2 编译与运行

```bash
# 运行全部测试
ceedling test:all

# 运行单个模块测试
ceedling test:test_voltage_sim

# 生成覆盖率报告
ceedling gcov:all

# 清理构建产物
ceedling clean
```
